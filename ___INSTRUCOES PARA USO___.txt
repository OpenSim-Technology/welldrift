INSTRUÇÕES PARA USO DO SOFTWARE WELLDRIFT V0.1:

Arquivos de entrada utilizados pelo programa:

Pasta default com resultados e dados de entradas: "\WellData\".

- WellData\Setup.txt	-> Arquivo com os dados iniciais default utilizados pelo programa.
- WellData\Inflow.txt 	-> Arquivo com a informação de entrada de massa de cada fase em cada completação.
--------------------------------------------------------------
ESQUEMA:
                         	POÇO
             ___________________________________________
            |   |     |     |     |     |     |     |   |
       HEEL x   |  x  |  x  |  x  |  x  |  x  |  x  |   x TOE
            |___|_____|_____|_____|_____|_____|_____|___|
                   ^     ^     ^     ^     ^     ^      ^
                   |     |     |     |     |     |      |  
			ENTRADA DE MASSA LATERAL

--------------------------------------------------------------

O arquivo "Inflow.txt" pode ser gerado a partir de uma planilha Excel simplesmente copiando os valores da planilha e colando-os no arquivo "Inflow.txt".

Arquivos de saída produzidos pelo programa:

- WellData\coordinates.txt 	-> Arquivos com as coordenadas x,y,z de cada ponto do poço.
- WellData\results.txt		-> Arquivo com as curvas de pressão, fração volumétrica e velocidade de cada fase
OBS: Os arquivos estão prontos para serem copiados e colados em uma planilha Excel caso o usuário queria uma análise mais avançada das curvas produzidas pelo programa.

Para saber mais sobre as equações e modelos utilizados no programa, ver referências do trabalho.

MODELOS DE FLUIDOS:

Óleo:
	-> Densidade constante
	-> Viscosidade constante
Água:
	-> Densidade constante
	-> Viscosidade constante
Gás:
	-> Cálculo da densidade		 
		rho = ref_rho + (Pressure - ref_Pressure)/a^2
		onde 'a' é a velocidade do som do fluido ref_rho 
		e ref_pressure são a densidade e pressão de 
		referência, respectivamente. Em geral, para gases,
		ref_rho = 0, ref_Pressure = 0 e a^2 = R*T_ref
		--------------------------------------------		
		ref_rho 	-> Densidade de referência do gás < Gas Ref. Density  >
		ref_pressure 	-> Pressão de referência do gás   < Gas Ref. Pressure >
		a 		-> Velocidade do som do gás	  < Gas Sound Speed   >
		
	-> Viscosidade constante
OBS: Os modelos utilizados para os fluidos podem ser alterados facilmente, bastando pequenas alterações no código fonte. Caso o usuário necessite de modelos mais complexos, entrar em contato pelo endereço de email do responsável pelo trabalho.
